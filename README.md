# E-commerce Sales Analysis Dashboard

## 📊 Project Overview
This project analyzes e-commerce sales data using Power BI. It provides insights into sales performance, customer behavior, and regional trends.

## 🚀 Features
- KPI Metrics (Total Sales, Profit, Orders)
- Monthly Sales Trend Analysis
- Category-wise Performance
- Region-wise Sales Distribution
- Interactive Filters (Slicers)

## 🛠 Tools Used
- Power BI
- SQL
